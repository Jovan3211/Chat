Direct Chat Manual

Here you will find instructions on how to use this program.

Connecting and Chat

IP

For the first text box, leave the IP address on default.

Before you connect, you must specify an IP address to the program, so the program knows where to connect to. Find out the persons IP address of who you want to talk to and then put that in the friend IP text box.

Port

You can enter any number between 1 and 65535. Your port number and the friend port number have to be different! In the 'You' port text box input any port number, but in the 'Friend' text box, you have to input the port number that the other user has put into his 'You' port text box.

Port forwarding should not be required, but if a connection is not established, try to port forward

Sending messages

To send a message, simply write some text in the bottom text box and press the 'Send' button or press enter.

Disconnecting

To disconnect, simply press the disconnect button from the file menu or right from the text boxes
